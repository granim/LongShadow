Welcome to my Spring boot application. This app is built with spring boot, JPA, Hibernate, Spring Security and is using a H2 in memory database. but also has configuration for a MySql database

For the heroku deploy, anytime the app is restarted any information created will be destroyed. This app also has CRUD operations implemented at the moment with more features to be added asap. You can create an account and then add coworkers and clients. A view that will sort clients and or coworkers by their timezone will be implemented soon.

Please open the login link to create an account if you want to test out the app.
